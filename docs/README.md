## Установка через Docker

* Скачиваем [Docker](https://www.docker.com/)
* Кланируем проект
* `CD Rest-client`
* `docker-compose build`
* Настраиваем `.env`
* `php init`
* `docker-compose up -d`


* Backend доступен по `http://localhost:21080/user/auth`
* Frontend по `http://localhost:20080/user/auth`
